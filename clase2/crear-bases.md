# Crear bases de datos en SQL

> Para crear una base de datos vamos a utilizar el comando **CREATE DATABASE**
 
> En la paleta de código escribiremos

    CREATE DATABASE nombre;  

> En este curso todos los ejemplos van hacer para una base de datos "introdb"

    CREATE DATABASE introdb;  

> Para ejecutar el código tenemos el ícono de "rayito" ⚡

## Eliminar una base 

> Para eliminar una base de datos vamos a utilizar el comando **DROP DATABASE**

> En la paleta de código escribiremos

    DROP DATABASE nombre;  

> Importantísimo: todo el código que yo no necesito ejecutar debo borrarlo.
> Si tengo en la imperiosa necesidad de guardar un código, me lo guardo en un archivo de texto pero no en la ventana de código.
